# RAG-Chatbot
1. Developed a custom LLM using Ollama, RAG architecture, and VectorDB, boosting response accuracy through optimized embeddings and real-time data integration.<br>
2. Enhanced retrieval speed and precision by refining similarity search algorithms and applying advanced NLP techniques.

# Things to install
1) Ollama Installation according to Operating System -> Install llama 3.2 with it.
2) pip install streamlit pymupdf langchain chromadb sentence-transformers langchain-ollama

# Steps to Run

1) Open Terminal
2) Run 'ollama serve'
3) Create an environmental variable 'venv' -> Install all the Dependencies i.e. with pip
4) Then run the main code using -> streamlit run main.py
